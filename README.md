# 🎮 ColdWar - Java Team Project

**ColdWar** es un juego desarrollado en Java como parte del proyecto grupal para el ciclo de **Desarrollo de Aplicaciones Multiplataforma (DAM)**. 

## 🚀 Características técnicas
Este proyecto se ha centrado en aplicar conceptos avanzados de **Programación Orientada a Objetos (POO)**, destacando:
* **Uso de Herencia:** Implementación de clases planeta y derivadas para la gestión de entidades del juego.
* **Lógica de juego:** Gestión de turnos/combate donde cada jugador decide a quien atacar.
* **Trabajo Colaborativo:** Desarrollado en equipo, aplicando metodologías de coordinación.
* **Diferentes modos de juego:** Combate en solitario contra bots o modo multijugador local.
  
## 🛠️ Tecnologías utilizadas
* **Lenguaje:** Java.
* **IDE:** VS Code / Eclipse.
* **Control de versiones:** Git.

## 🕹️ Cómo ejecutar el juego
* **1-Clona el repositorio.**
* **2-Abre el proyecto en tu IDE preferido (VS Code / Eclipse / IntelliJ).**
* **3-Ejecuta la clase Main.java**
